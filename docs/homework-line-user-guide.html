Homework Demo 登記與預約說明

第一次使用的同學，請先完成 Homework 資料登記；之後就可以在 LINE 上選助教、日期與時段。 如果你已經有預約，也可以用同樣的方式查詢、取消，或申請逾期取消。

先登記 先輸入學號與姓名，建立 Homework 資料。
再預約 依序選助教、日期、時段，完成預約。
可查詢 / 取消 若臨時無法出席，也可以依規則取消或申請逾期取消。

開始前請先看

必填資料 請輸入學號與姓名 格式為 `<學號> <姓名>`，例如 `114106123 王小明`。
預約流程 共 3 個步驟 選助教、選日期、選時段，按下後即完成預約。
取消規則 時限內可直接取消 若已超過時限，就要改走逾期取消申請。
結果通知 逾期取消要等審核 送出申請不代表已取消，請等助教審核結果通知。
